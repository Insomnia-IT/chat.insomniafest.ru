Пошаговая Инструкция

  1. Скачай Element

    Есть приложения для Android, iOS, Windows, macOS или Linux, или используйте веб-версию на chat.insomniafest.ru (работает в любом браузере, но в мобильных браузерах может быть ограничена функциональность).

  2. Зарегистрируйся в нашем ТГ боте (пока телега еще работает)

    Бот для регистрации: @insomnia_chat_registration_bot. Важно: обязательно скажите своему HR-менеджеру добавить вас в список волонтеров 2026. При входе в Element нужно выбрать опцию "другой домашний сервер" и указать insomniafest.ru (вот так). После входа сразу смените пароль.

  3. Про сквозное шифрование (или как не потерять историю сообщений)

    В нашем чате можно создавать группы (они тут называются комнатами) с шифрованием и без. Ключи шифрования хранятся на ваших устройствах. Это означает, что если вы потеряете единственное устройство, с которого вы заходите в чат, вы потеряете доступ к зашифрованным сообщениям. Этого можно лего избежать: в настройках шифрования нужно разрешить "хранение ключей" и обязательно сохранить ключ восстановления (по-сути это еще один пароль) в безопасном месте (ведь уже все пользуются менеджерами паролей?). Второй вариант - залогиниться на двух (или больше) устройствах одновременно (тогда новое устройство получит ключи от старого).

Как устроены Matrix и Element

  • Matrix это протокол и сервер (у нас свой сервер).
  • Element это клиентское приложение (как браузер для Matrix).
  • Комнаты это чаты: по направлениям, службам, локациям.
  • Можно войти на нескольких устройствах под одним аккаунтом.

Чем это отличается от привычных мессенджеров

  • Данные и учетные записи живут на нашем сервере.
  • Можно использовать разные клиенты, не только Element (они должны быть совместимы с протоколом Matrix).
  • Сквозное шифрование (ключи на ваших устройствах).
